# Repository Dashboard

A Next.js application for managing and monitoring code repositories with file analysis capabilities.

## Features

- 📊 **Repository Dashboard**: View all repositories in a clean grid layout
- 🔍 **Search & Filter**: Search repositories and filter files by language and criticality
- 📁 **File Management**: Browse files within each repository with detailed metadata
- 🛡️ **Security Insights**: Identify sensitive files and risk levels
- 📈 **Repository Statistics**: Quick overview of files, languages, and risk assessment
- 🎨 **Responsive Design**: Works seamlessly on desktop and mobile devices

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: Zustand
- **Data**: Local JSON file
